# Vehicle-to-Vehicle (V2V) Communication System 

This Arduino-based V2V communication system is designed for accident prevention, obstacle detection, and speed monitoring using wireless NRF24L01, GSM alerts, and onboard diagnostics.

##  Features
- Real-time speed measurement via RPM sensor
- Obstacle detection using ultrasonic sensor
- Wireless data exchange via NRF24L01
- GSM alerts to phone + HTTP POST to web dashboard
- LCD interface for live feedback
- Servo steering & motor control
- Diagnostic checks on startup

## Hardware Used
- Arduino Nano / Uno
- GSM Module (SIM800L)
- NRF24L01 wireless module
- LCD I2C 16x2
- HC-SR04 Ultrasonic sensor
- RPM sensor
- Motor driver (L298N)
- Servo motor
- Buzzer
- Push buttons (forward/reverse)
- Power supply & wiring
